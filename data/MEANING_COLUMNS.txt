

STATE	--> Name of the State
CD_MUN --> Electoral code of the municipality
NM_MUN --> Name of the municipality
VOTES_INCUMBENT --> Number of votes of the incumbent candidate (since 2002, it's always been the PT - 13)
VOTES_CHALLENGER --> Number of votes of the challenger candidate (from 2002 to 2014, it had always been the PSDB - 45)
QT_ABST --> Number of voters that did not show up in the day of the elections (A)
QT_APT --> Number of voters registered to vote in the municipality (B)
QT_ATTEND --> Number of voters that showed up in the day of the election (C = B - A)
QT_BLANK --> Number of blank votes
QT_NULL --> Number of null votes
